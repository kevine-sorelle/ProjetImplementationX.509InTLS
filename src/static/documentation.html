Certificate Validator - Documentation

Introduction

Certificate Validator est une application web basée sur Flask conçue pour générer, valider et visualiser des certificats X.509 incluant des clés publiques de mécanisme d'encapsulation de clé (KEM).

Fonctionnalités

  • Générer des certificats X.509 avec des clés publiques KEM
  • Extraire et afficher les détails des certificats
  • Encapsuler et décapsuler des clés secrètes partagées
  • Télécharger les certificats générés

Installation

        git clone https://github.com/kevine-sorelle/ProjetImplementationX.509InTLS.git
        cd certificate-validator
        pip install -r requirements.txt
        python app.py

Endpoints API

  • GET / - Afficher les détails du certificat et les opérations KEM
  • GET /download-cert - Télécharger le certificat X.509 généré

Utilisation

Une fois l'application en cours d'exécution, ouvrez http://127.0.0.1:5000/ dans votre navigateur pour visualiser et gérer les certificats.
